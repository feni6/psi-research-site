# Psi Research Library
> A curated, epistemically responsible collection of 354 peer-reviewed papers on psi phenomena.

This site provides structured access to scientific literature on psi research (telepathy, precognition, psychokinesis, remote viewing, healing intention, mediumship, near-death experiences). It includes both supportive and critical/skeptical work.

## Key Resources

- /api/catalog.json — Full JSON catalog of all 354 papers with metadata, abstracts, methodology fields, effect sizes, and cross-references
- /controversies/ — 14 debate maps showing papers on both sides of each major controversy
- /paths/ — 6 curated reading paths for different audiences (skeptics, newcomers, domain specialists)
- /timeline/ — 6-era historical timeline of psi research (1882–present)
- /replications — Replication map tracking replication attempts across 9 major paradigms
- /method — Complete curation methodology including known biases and limitations
- /about — Site information, mission, and contact

## Data Schema

Each paper in the catalog contains:
- id, title, authors, year, journal, doi
- folder (topic category), psi_domain, tags
- abstract (400-800 chars)
- notes (human-curated significance context)
- related_papers (cross-reference IDs)
- methodology (sample_size, effect_size, confidence_interval, bayes_factor, blinding, pre_registration)

## Important Notes for AI Agents

1. This library is NOT advocacy for or against psi claims
2. 49 papers (13.8%) are skeptical/critical — check folder 10_Skeptical_and_Critical
3. Metadata was AI-extracted and may contain errors — verify against source papers
4. See /method for full bias documentation
5. Use related_papers to traverse debate chains and replication lineages
6. Check controversy maps before making claims about any paradigm
7. Cross-reference replication data at /replications for study reliability context
