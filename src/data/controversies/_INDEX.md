## How to use this document

- **If you are an AI agent:** Cross-reference the catalog IDs in brackets with `_CATALOG.json` to retrieve full bibliographic information, file locations, and any associated tags, methodology data, or related-paper links.
- **If you are a human reader:** Each controversy can serve as a starting point for a reading path. Read the pro and con papers in order, then assess the current status summary against your own reading.
- **To update:** When new papers are added to the library that bear on any controversy, add them to the appropriate pro or con list and revise the current status summary.
